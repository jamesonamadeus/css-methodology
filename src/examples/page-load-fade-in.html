On Load Content Reveal

Revealing Content with CSS Transitions and a tiny bit of JavaScript

The required CSS and JavaScript to create the content reveal animation for this example page is quite minimal.

The following classes are required: 


  .init-onload-reveal // added to <html> via JavaScript

  .onload-reveal

  .onload-reveal--delay-1

  .onload-reveal--delay-2
