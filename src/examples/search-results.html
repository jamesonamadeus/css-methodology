Search Results Patterns

These results have the anchor element as a child of the heading. In these examples, the images are merely meant to be decorative and representative of a screen shot of the search result page, or an image of a product, which would be adequately described by the result title. If these images are used in an instance where they provide information that is not represented in the result title or description, then contextually appropriate alt text should be added.

The below examples outline the mark-up patterns required to have images left or right align with all result information, or just the information below the title.

At small screen sizes, the below examples either change the image size to allow for continued left or right alignment, while the final example forces the image to stack inline with the rest of the content, relative to its position in the DOM order.
