Silent Classes

Classes that are only instantiated if they are invoked by @extend in normal CSS classes.

Many of these are being used in base elements to normalize properties that are repeated among various selectors.

  • %reset--spacing — margin and padding 0
  • %reset--border — border and outline 0
  • %help — visual styling for elements with helpful [title] text

Mixins
