Date Inputs

Date inputs are styled like normal text inputs, but some browsers have fully implemented these inputs, giving them additional functionality beyond normal text inputs.

Since these inputs are not consistent across all browsers, it may make sense to not use these inputs just yet, or if you do use them, use them in conjunction with a plug-in/polyfill for the browsers that don't support their native functionality.

Color Input

The color input is not fully supported in all browsers yet. For browsers that don't support the color input, it will fall back to a normal text input styling.
