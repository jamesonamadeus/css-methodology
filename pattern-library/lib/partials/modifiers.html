Width Classes

Use these classes to create grid columns, or to give specific elements specific percentage based widths.

  • .width-10
  • .width-20
  • .width-25
  • .width-30
  • .width-33
  • .width-40
  • .width-50
  • .width-60
  • .width-66
  • .width-70
  • .width-75
  • .width-80
  • .width-90
  • .width-100

Width classes without suffixes will stay their designated width at all screen sizes. The following suffixes will set the designated width at only the specified breakpoint and above.

  • --sm
  • --portrait
  • --md
  • --lg
  • --1024

Example:

.width-40
.width-40--sm
.width-40--portrait
.width-40--md
.width-40--lg
.width-40--1024
.width-60--portrait.width-40--1024
