Flex Grids

The base flex box grid. Utilizes -left margin/ +left padding to create column gutters.

Note: flexbox grids are not truly meant for full page layouts. A standard two column layout may be better suited to use a float-based grid until the CSS grid layout properties are finalized.

Note: use only what you need for float and flexbox grids. Remove (comment out) unused width or modifier classes so as to not add unnecessary bloat to your compiled CSS.

Flex Grid: Space Between

This Flex Grid helper removes the default negative margin/positive padding gutter setup, and instead uses flexboxe's property justify-content: space-between to create gutters based on 'free space' from smaller column sizes.
