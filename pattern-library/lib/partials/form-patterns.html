Forgot Password Input Pattern

Forgot password links should be placed last in the DOM order, with the input label, input, and any error message appearing before it.

The forgot password link is often positioned to the top/right of the element, and that can be done via the .link--forgot-pw class being added to the link element (or if applicable, it's direct parent container).
