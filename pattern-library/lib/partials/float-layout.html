Float Helpers

Use floats to align content to the left or right of a content area.

To ensure that a container doesn't collapse in on itself, due to its floating children, appropriately apply a class of .clearfix or .overflow-clear to the parent container of the floating elements.

Listing of float helper classes:

  • .float-left, .float-left--sm
  • .float-right, .float-right--sm
  • .float-left--spacing
  • .float-right--spacing
  • .float-left--breakout
  • .float-right--breakout
  • .clearfix
  • .clear-both
  • .overflow-clear

Float Grids

In instances where using flexbox would be overkill, or it's not well supported (older browsers), use a float-based grid.

See the modifier classes > width modifiers for the full list of available classes that double as columns with the float-grids.

Note: use only what you need for float and flexbox grids. Remove (comment out) unused width or modifier classes so as to not add unnecessary bloat to your compiled CSS.
