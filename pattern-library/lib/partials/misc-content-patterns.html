Semantic Content Shifts (hr)

Appropriate for when a topic shifts within long-form text. The same semantic meaning as a <section> followed by another <section>.

Light (classless) and dark (.hr--inverse) to allow for visibility on light or dark backgrounds.

Load More

Load more buttons are utilized in place of (or possibly in conjunction with) pagination.

Triggering a load more button should reveal the next set of information, and ensure focus is moved to the new area, while removing the activated load more button / it's container from the DOM.
